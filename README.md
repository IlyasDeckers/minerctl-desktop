# minerctl

> Minerctl desktop application

#### Requirements
Claymore v10.0.0 has to be located in your C drive `C:\Claymore_v10.0`

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build MinerC.TL application for production
npm run build

```
